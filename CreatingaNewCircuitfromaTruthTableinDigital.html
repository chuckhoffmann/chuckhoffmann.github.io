Creating a New Circuit from a Truth Table in Digital

Creating new circuits from truth tables is possible in Digital by using the Table editor window. In this window, you can add, delete, or rename input variables and output columns, as well as set the desired output results.

To access the Table editor window, select the “Synthesis” item from the “Analysis” menu. A window allowing you to edit the truth table will appear.

The gray-shaded columns on the left represent the inputs to the truth table, while the columns on the right represent the outputs.

Setting Values in Output Columns

You cannot change the values in the input columns: the values are automatically inserted by Digital.

To change the values in the output columns, left-click on the value you want to change. A drop-down box with the selectable values 0, 1, and X will appear.

Output Value Drop-Down

If the list is not shown, click the downward-pointing triangle to display it.

Renaming, Adding and Moving Input Variables and Output Columns

You can rename the variable and columns in your truth table by right-clicking on the column heading. This will bring up a small window titled “Attributes”, with the single text input field “Label:”.

Column Attributes

You can use tildes to add negation overlines, underscores to add subscripts, and carets to add superscripts to your labels. Subscripts and superscripts will not be displayed in the column heading, and tildes will be converted to logical not symbols. However, all of these will behave normally in the final circuit.

Representations of negation, subscripts and superscripts.

To add a new variable or column, select either the “Add Input Variable” or “Add Output Column”, as appropriate, from the “Edit Menu”.

To move columns you can either click and drag the columns to arrange them, or select either the “Reorder/Delete Input Variables” or the “Reorder/Delete Output Columns” item from the “Edit” menu. This will bring up the appropriate “Reorder/Delete...” window..

Reorder/Delete Input Values

Creating the Circuit

Once you have set all your input values and output columns, you can create your circuit by selecting one of the options from the “Create” menu. Remember also to save your truth table

As an example, here is the truth table for a simple 2 to 4 decoder with an enable line.

Example Truth Table for 2-to-4 Decoder with Enable

In this table, the input variable representing the enable line is labeled “en”, while the two input variables representing the selector lines are labeled “I_1” and “I_0”. (Remember that superscripts and subscripts are not shown in table headings, but will render correctly in the final circuit.) Three more output columns have been added and all have been relabeled as O3 to O0.
